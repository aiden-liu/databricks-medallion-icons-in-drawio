# databricks-medallion-icons-in-drawio
Databricks medallion architecture icons, exported from drawio scratchpad.

## How to use
Create a drawio canvas, in the scratchpad, click edit then import.

## RGB INFO
- BRONZE: Dulux Olympic Bronze (paint shade): RGB (115, 85, 41) or #735529.
- SILVER: Metallic Silver: RGB (167, 167, 173) or #A7A7AD
- GOLD: Olympic Gold: (216, 159, 57) or #D89F39
